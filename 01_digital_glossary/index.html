HTML Tag Glossary

TagDescription
<h1>Heading level 1 - largest heading
<p>Paragraph - defines a paragraph of text
<a>Anchor - creates a hyperlink
<img>Image - embeds an image
<div>Division - a block container element
<span>Span - an inline container element
<table>Table - creates a data table
<ul>Unordered list - bullet point list
<ol>Ordered list - numbered list
<li>List item - item inside ul or ol
<form>Form - creates an HTML form
<input>Input - creates an input field
